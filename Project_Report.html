Campus Connect – Project Report

1. Introduction

Campus Connect is a modern, responsive web platform designed to showcase campus events, activities, and announcements in an interactive and user-friendly manner. It allows students and faculty to stay updated with upcoming events, view galleries, and contact organizers easily.

Objectives:

  • Provide a central hub for campus events
  • Increase student engagement
  • Enhance user experience with dynamic content and interactivity

2. Features Implemented

Feature Description
Responsive Navigation Mobile-friendly hamburger menu with smooth scrolling
Homepage Hero & About Preview Hero section with "Explore Events" button
Dynamic Event Cards Upcoming events loaded dynamically using JavaScript
Gallery Carousel Interactive carousel with prev/next buttons & auto-slide
Contact Form Validation Alerts user if any field is empty before submission
Enhanced Interactivity Hover effects, smooth scrolling, dynamic content updates
UI/UX Design Consistent layout, responsive across all devices
Deployment Fully functional live site on GitHub Pages

3. Technologies Used

  • HTML5 – Structure of web pages
  • CSS3 – Styling, layout, responsive design
  • JavaScript – Dynamic content, form validation, carousel, interactivity
  • GitHub Pages – Hosting and live deployment
