# Campus-Connect-Smart-Event-Activity-Management-Platform
Campus Connect is a responsive Campus Event Management Website developed using HTML, CSS, and JavaScript. The project showcases campus events, activities, gallery, and contact information with interactive and responsive UI. Built as part of an MCA Applied Learning Project.


# Campus Tech Fest Carousel

## Project Description
This is a responsive web project showcasing a **college tech fest image carousel** for Stage 2 submission. The carousel highlights multiple events at Jain University Bangalore and includes interactive features with smooth transitions and captions.

## Features
1. **Image Carousel** - Displays 4 images with headings and descriptions.
2. **Next/Previous Buttons** - Navigate through slides manually.
3. **Auto-Slide** - Carousel automatically moves every 3 seconds.
4. **Captions** - Each image has a heading and description.
5. **Responsive Design** - Works on mobile, tablet, and desktop screens.

## Technologies Used
- HTML
- CSS
- JavaScript

## Instructions
1. Open `index.html` in a browser to view the carousel.
2. Use the left/right buttons or wait for the auto-slide to see all images.
3. All images are stored in the `images/` folder.
